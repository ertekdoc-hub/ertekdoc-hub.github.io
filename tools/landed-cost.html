How it works

MVP – indicative only. Always verify with your customs broker.

Inputs

  • Destination: country VAT rate preset (NL 21%, DE 19%, BE 21%, FR 20% …)
  • Goods value (€): ürün bedeli (FOB/EXW + mal bedeli)
  • Freight + insurance (€): nakliye + sigorta
  • HS code (first 4 digits): ör. 3923, 4202, 6109, 6204, 6403, 8517
  • Clearance fee (€): gümrük/komisyon
  • Units: opsiyonel (birim başı maliyet için)

Calculation

  1. CIF = Goods value + Freight + Insurance
  2. Duty = CIF × duty rate
    Presetler: 3923 ≈ 6.5%, 4202 ≈ 9%, 6109/6204 ≈ 12%, 6403 ≈ 17%, 8517 = 0%. (HS girilmezse “default” oran kullanılır.)
  3. VAT base = CIF + Duty + Clearance fee
  4. VAT = VAT base × destination VAT rate
  5. Landed cost = CIF + Duty + VAT + Clearance fee
  6. Eğer Units girilmişse: Cost / unit = Landed cost ÷ Units

Notes

  • Oranlar yaklaşık MFN/EU örnekleri; anti-dumping, ek vergi, menşe ve anlaşmalar sonucu değişebilir.
  • FOB/CIF farklılıkları için forwarder/gümrükçünüzle teyit edin.
